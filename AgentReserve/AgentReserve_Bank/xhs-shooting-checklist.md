# AgentReserve 拍摄执行清单

> **场景**: 黑客松现场 / 酒店房间 / 任意有桌子和电源的地方
> **总时间预算**: 拍摄 1-1.5 小时 + 剪辑 1.5-2 小时 = **3-3.5 小时完成**
> **最低配置**: 1 个人 + 1 台手机 + 1 台电脑

---

## 一、设备清单

### 必带设备（没有就借，借不到想办法）

| 设备 | 用途 | 最低要求 | 理想配置 |
|------|------|---------|---------|
| **手机** | 拍人像段落 | 近 3 年的智能手机即可 | iPhone 12 / 安卓旗舰，后置摄像头 |
| **手机支架/三脚架** | 固定机位，解放双手 | 任何能立住的东西（水杯+书堆） | 便携三脚架（京东 30 块，或向其他队借） |
| **电脑** | 屏幕录制 + 剪辑 | 能跑产品 Demo 的电脑 | MacBook Pro / 高性能Win本 |
| **耳机/麦克风** | 收音（环境吵时必备） | 手机原装耳机麦 | 领夹麦（借一个，或 AirPods） |
| **补光** | 让脸能看清 | 台灯、屏幕白光、窗户自然光 | 环形补光灯（向其他队借） |

### 可选设备（有就带上，没有算了）

| 设备 | 用途 | 替代方案 |
|------|------|---------|
| 第二台手机 | 同时录备份 / 拍花絮 | 让队友拿自己手机帮你录 |
| 外接显示器 | 展示产品界面更大更清晰 | 直接用笔记本屏幕 |
| 无线领夹麦 | 收音更清晰 | 有线耳机麦 / 安静环境下手机直录 |
| 充电宝 | 防止手机没电 | 插电源拍 |

### 现场借装备话术（如需）

## 二、拍摄场景建议

### 场景选项对比

| 场景 | 优点 | 缺点 | 适用情况 |
|------|------|------|---------|
| **黑客松现场工位** | 真实、有氛围、背景自然 | 可能很乱、噪音大、光线不稳定 | ⭐ 首选，最有"极限挑战"感 |
| **酒店房间/会议室** | 安静、可控、光线稳定 | 缺少黑客松氛围 | 现场太乱时的备选 |
| **咖啡桌/休息区** | 光线好、背景干净 | 可能有人走动、不够硬核 | 想要"优雅"风格时 |

### 推荐场景：黑客松现场（首选）

**布置清单**:
- [ ] 找一个相对干净的角落（或者把桌面的杂物推到一边）
- [ ] 背后最好有**电脑屏幕发光**+**便利贴/白板**作为背景元素
- [ ] 如果太乱，用**大光圈/人像模式**虚化背景
- [ ] 确保头顶没有死亡顶光（用台灯补光或调整位置）

**快速收拾桌面（2分钟）**:
```
1. 把零食、空杯子拿掉
2. 保留：电脑（屏幕亮着）、键盘、1-2个有设计感的物品（如耳机、纸质笔记）
3. 背景：如果有团队名牌/赛事标识，露出来
```

## 三、人像段落拍摄指南

### 谁出镜？

| 角色 | 建议 | 原因 |
|------|------|------|
| **主讲人** | 对产品最熟悉、表达最清晰的队员 | 讲错可以重拍，不熟悉的NG次数多 |
| **替补** | 如果主讲人镜头恐惧，换第二熟悉的人 | 宁可换人也不要拍一个僵硬的视频 |

### 穿着建议

| ✅ 推荐 | ❌ 避免 |
|---------|---------|
| 纯色 T 恤 / 连帽衫（黑/白/灰/藏青） | 条纹、格子、logo 太多的衣服 |
| 有科技感的配件（耳机挂脖子上、智能手表） | 正装、过于休闲（睡衣风） |
| 团队统一元素（如有团队 T 恤） | 颜色太跳、与背景撞色 |

### 站位与眼神

| 项目 | 要求 |
|------|------|
| **站位** | 距离镜头 0.8-1.2 米，半身或胸像 |
| **眼神** | **看镜头后方 5cm 处**（模拟看观众眼睛），不要看屏幕也不要飘忽 |
| **角度** | 镜头略低于眼睛高度（不要俯拍），手机架在桌面或支架上 |
| **手势** | 说到"加油站"时左手加油手势，"信用卡"时右手刷卡手势，动作要大 |
| **表情** | 开场严肃/紧张（钩子），中间自然（讲解），结尾自信微笑（CTA） |

### 拍摄技巧（用手机也能拍出质感）

| 技巧 | 操作方法 |
|------|---------|
| **背景虚化** | 打开人像模式 / 电影效果模式 |
| **画质最高** | 相机设置 → 分辨率 4K 30fps（不必 60fps，省空间） |
| **曝光锁定** | 长按屏幕锁定曝光，防止光线变化时脸忽明忽暗 |
| **分段拍** | 不要一次录 3 分钟！每段 20-30 秒单独录，后期拼接。说错就重来，只浪费 20 秒 |
| **每条拍 2 遍** | 同一内容拍两条，选最好的那条，防止唯一素材翻车 |

### 人声段落分段录制清单

| 段落 | 时长 | 内容提示 | 拍摄次数 |
|------|------|---------|---------|
| 黄金3秒钩子 | 5秒 | "你的 AI Agent 做到一半，Token 没了。" | 拍 3 遍，选最炸的 |
| 痛点共鸣 | 15秒 | 摊手+摇头 | 拍 2 遍 |
| 产品介绍 | 30秒 | 手势配合 | 拍 2 遍 |
| 价值升华 | 20秒 | 语气慢、稳 | 拍 2 遍 |
| 收尾金句+CTA | 20秒 | 自信微笑 | 拍 3 遍，选最好的 |

> **人像总拍摄时间**: 约 30-40 分钟（含试拍、NG、重拍）

## 四、屏幕录制指南

### 录制软件推荐

| 平台 | 推荐软件 | 下载/使用方式 | 特点 |
|------|---------|-------------|------|
| **Mac** | QuickTime Player（自带） | 启动 → 文件 → 新建屏幕录制 | 免费、稳定、画质好 |
| **Mac** | OBS Studio | obsproject.com | 免费、功能全、可高亮鼠标 |
| **Windows** | OBS Studio | obsproject.com | 免费、功能全 |
| **Windows** | Xbox Game Bar（自带） | Win+G 启动 | 无需安装，快速启动 |
| **跨平台** | Screen Studio（付费） | screen.studio | 自动缩放鼠标位置、最省力 |

### 录制设置

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **分辨率** | 1920×1080（1080p） | 小红书/B站都支持，清晰度够 |
| **帧率** | 30fps | 屏幕录制不需要 60fps，省空间 |
| **编码格式** | H.264（兼容性最好） | 几乎所有剪辑软件都支持 |
| **鼠标高亮** | **必须开启** | 让观众看清你点哪里 |

### 鼠标高亮工具

| 平台 | 工具 | 设置 |
|------|------|------|
| Mac | Cursor Pro / Mouseposé | 鼠标周围加光圈 |
| Windows | Mouse Highlight / PointerFocus | 点击时加涟漪效果 |
| 通用 | OBS 插件 | 在 OBS 里加鼠标追踪滤镜 |

### Demo 录制操作步骤

**Demo 1：应急借贷**
```
1. 打开产品界面，进入 Dashboard
2. 找到"翻译 Agent"任务，确认已执行到 73%
3. 开始录屏
4. 展示 Token 余额告警（红色提示）
5. 点击"一键借贷"按钮（停顿 0.5 秒）
6. 展示"借款成功"提示 + Token 到账动画
7. 展示任务继续执行，进度条推进到 100%
8. 展示"任务完成"状态
9. 停止录屏
10. 回放检查：鼠标是否清晰？流程是否连贯？
```

**Demo 2：预估+智能借贷**
```
1. 打开"新建任务"界面
2. 输入任务描述："分析 1000 篇论文并生成综述"
3. 点击"预估成本"
4. 展示数字跳动动画（从 0 到 5000 Token）
5. 展示"确认借款"按钮，点击
6. 展示借款成功，任务自动开始
7. 展示进度条推进
8. 停止录屏
```

> **录屏总时间**: 约 20-30 分钟（含准备、录制、检查）

## 五、剪辑建议

### 推荐剪辑工具

| 工具 | 平台 | 难度 | 特点 | 推荐场景 |
|------|------|------|------|---------|
| **剪映** | 手机/电脑 | ⭐ 极易 | 字幕自动识别、BGM库丰富、小红书模板多 | ⭐ **首选**，最适合这个任务 |
| **CapCut** | 手机/电脑 | ⭐ 极易 | 剪映国际版，功能和剪映一样 | 海外队友用 |
| **iMovie** | Mac/iOS | ⭐⭐ 易 | 苹果自带，稳定 | Mac 用户快速剪辑 |
| **Premiere** | 电脑 | ⭐⭐⭐ 难 | 专业级，功能全 | 如果有会 PR 的队员 |

### 剪映具体操作流程（推荐）

**Step 1：导入素材（5分钟）**
```
1. 打开剪映 → 开始创作
2. 导入所有人像视频片段 + 屏幕录制片段
3. 按脚本顺序拖入时间轴
```

**Step 2：粗剪（20分钟）**
```
1. 每个片段开头结尾各留 0.5 秒空白（方便加转场）
2. 删除所有 NG 片段、停顿、咳嗽、口头禅（"呃""那个"）
3. 调整片段顺序，确保和脚本一致
4. 片段之间加转场："闪黑"或"叠化"，0.3 秒即可
```

**Step 3：加字幕（15分钟）**
```
1. 点击"文字" → "识别字幕" → "开始识别"
2. 自动识别后，人工检查错字（科技名词容易识别错）
3. 重点词高亮：选中关键词 → 改颜色为黄色/橙色
4. 字幕样式：
   - 字体：思源黑体 Bold / 抖音体
   - 大小：根据画面调整，手机上看清楚
   - 描边：黑色描边 3px（确保任何背景都清晰）
   - 位置：画面下方 1/4 处，不要挡到脸或操作按钮
```

**Step 4：加 BGM（10分钟）**
```
1. 剪映音乐库搜索：
   - 钩子段："紧张""电子""科技感"
   - Demo段："轻快""节奏"
   - 收尾段："宏大""史诗"
2. 或者直接用推荐 BGM（见脚本文件）
3. 音量调整：BGM 主轨道 -15dB，确保人声清晰
4. 在段落切换处做 BGM 过渡（淡入淡出）
```

**Step 5：加特效（10分钟）**
```
1. 关键按钮点击处：加"缩放"或"高亮"特效
2. 数字跳动处：加"计数器"动画（剪映贴纸里有）
3. 产品 Logo 页：结尾加"黑底白字"标题卡
4. 赛事标签页：结尾 3 秒全屏显示 #南客松 #Flux南客松S2
```

**Step 6：导出（5分钟）**
```
1. 分辨率：1080p
2. 帧率：30fps
3. 码率：默认即可（小红书会二次压缩）
4. 格式：MP4
5. 文件名：AgentReserve_南客松_v1.mp4
6. 导出后立刻播放检查一遍！
```

> **剪辑总时间**: 约 1.5-2 小时（用剪映）

### 加速剪辑的技巧

| 问题 | 解决方案 |
|------|---------|
| 没时间精剪 | 剪映"智能剪口播"功能，自动剪掉停顿和重复 |
| 字幕不想手调 | 先用自动识别，只改明显错误，不追求 100% 精准 |
| BGM 找不到合适的 | 剪映搜"科技""电子"，前 3 首随便选一个，差别不大 |
| 转场不想一个个加 | 全选片段 → 批量加"叠化"转场 |
| 画质导出太慢 | 用手机版剪映导出，比电脑版快（压缩率高一点但够用） |

## 七、常见问题急救

| 问题 | 急救方案 |
|------|---------|
| **产品界面还没做好** | 用 Figma/设计稿代替录屏；或用静态图+放大动画模拟操作流程 |
| **主讲人镜头恐惧症** | 换人；或只拍手+声音出镜；或用卡通头像代替 |
| **环境太吵无法收音** | 去楼梯间/走廊/洗手间（关风扇）录人声，回现场录画面 |
| **手机存储空间不够** | 分段拍，每段拍完立刻 AirDrop/微信传给别人备份，然后删本地 |
| **没时间剪辑** | 用剪映"一键成片"功能，导入素材后 AI 自动配 BGM 和转场，人工只调字幕 |
| **录屏画质模糊** | 检查录制分辨率是否为 1080p；检查产品界面缩放是否为 100% |
| **导出后文件太大** | 用 HandBrake 压缩，或直接上传让平台自动压缩 |
| **视频审核不过** | 检查是否有敏感词；去掉"借贷"改用"预支""加油"等替代词重新导出 |

## 八、一句话总结

```
最低配置：1 人 + 1 手机 + 1 电脑 + 1 小时 = 能交差
推荐配置：2 人分工（1 人拍+1 人剪）+ 2.5 小时 = 出精品
理想配置：3 人（主讲+拍摄+剪辑并行）+ 1.5 小时 = 又快又好
```

**核心原则**：先完成，再完美。赛事截止前能交出一个完整视频就是胜利。
