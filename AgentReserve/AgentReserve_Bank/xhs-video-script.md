# AgentReserve 小红书视频脚本（完整版 2分30秒）

> **项目**: AgentReserve — Agent Network 的首个流动性基础设施
> **人设**: 年轻的 AI 开发者，刚经历 18 小时极限黑客松开发
> **风格**: 科技但不晦涩，用「加油站+信用卡」替代金融术语
> **标签**: #南客松 #Flux南客松S2

## 【字幕样式建议】

| 元素 | 建议 |
|------|------|
| **字体** | 粗体无衬线（如思源黑体 Bold / 抖音默认字幕体） |
| **颜色** | 白字黑边（确保在任何背景上都清晰） |
| **关键词高亮** | 核心词用 **黄色/橙色** 高亮，如"Token 没了""一键借贷""5秒到账" |
| **表情包** | 适当加 emoji（🚨⚡💰🔄）增强小红书感，但别超过每句1个 |
| **出现时机** | 比台词提前 0.2 秒出现，延后 0.3 秒消失，方便阅读 |
